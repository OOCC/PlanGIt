Stack Pointer(register)
	wiki
	A stack register is a computer central processor register whose purpose is to keep track of a call stack。
	（keep track of 记录）

	whatis.techtarget.com:
	A stack pointer is a small register that stores the address of the last program request in a stack.A stack is a         specialized buffer which stores data from the top down. As new requests come in, they "push down" the older ones. 

	key words: register，address，last program request

	register是集成电路中非常重要的一种存储单元，it looks like a buffer，常用于保存：数据，地址，指令。
