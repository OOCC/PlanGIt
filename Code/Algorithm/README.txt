思路

	设1-100之间共有m个这样的数，1-n之间则有f(1-n)个这样的数。设n的低于100的数为xy，f(xy)表示1-xy之间满足条件的数的个数。
		分两种情况讨论：

			当n的最高位是1：
				
				f(1-n) = n/100*m + xy
			
			当n的最高位是不1：

				f(1-n) = n/100*m + f(xy)
